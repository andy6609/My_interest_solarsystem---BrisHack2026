ChatGPT Data Export (Sample)
============================

이 ZIP은 '데이터 내보내기' 결과가 어떻게 생겼는지 연습/테스트용으로 만든 샘플입니다.

포함 파일
- chat.html: 사람이 보기 쉬운 HTML 형태의 대화 기록(샘플)
- conversations.json: 기계가 읽기 쉬운 JSON 형태의 대화 기록(샘플)
- user.json: 계정/설정 관련 샘플 데이터
- README.txt: 안내문

활용 예
- chat.html을 브라우저로 열어 검색(Ctrl+F)
- conversations.json을 파싱해서 Markdown/CSV/노션 형태로 변환
